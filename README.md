# aworld-client

## Install

```
$ pip install -r requirements.txt
```

## Play

```
$ python main.py
```

## Buttons

| Keyboard |        Action        |
| -------- | -------------------- |
| Left(←)  | Rotate to left       |
| Right(→) | Rotate to right      |
| Up(↑)    | Move forward         |
| Down(↓)  | Move backward        |
| Space    | Pick up item         |
| a        | Attack               |
